# Glossário (EXPANSÃO360)

Termos fixos usados no projeto para manter consistência entre domínio, CLI e Web.

## Camadas
- registry → cadastro
- workflow → execução (fluxo de trabalho)
- operation → operação

## Entidades/Conceitos
- location → loja
- mount → instalação
- kit → kit
- item → item
- card → chamado 

## Identificadores
- ativo (asset tag) → ativo
- serial → série
- IP → IP

## Atributos
- configurable → configurável
